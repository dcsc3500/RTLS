## RTLS (2018.01.01 ~ 2018.02.28)
소켓프로그래밍 공부를 위한 프로젝트

>참고 : [ppt](https://github.com/dcsc3500/RTLS/tree/master/ppt) 


### 사용언어
* Java
* Java Swing(For GUI)
* Maria DB(DB설정을 다시 해줘야함)

### 기능
1.Monitoring Client
  - User가 위험구역에 들어가면 알림
  - 클라이언트와의 채팅
  - 특정 User가 다닌 길 추적
  
2.User Client
  - 클라이언트와의 채팅
